# Security Policy

## Reporting a Vulnerability

AgentShield is a security tool. We take vulnerability reports
seriously and respond within 48 hours.

**DO NOT open a public GitHub issue for security vulnerabilities.**

Report privately to: belhekaraditya96@gmail.com
Subject line: [AgentShield SECURITY] Brief description

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge within 48 hours and provide a timeline
for the fix. We credit all reporters in the CHANGELOG.
